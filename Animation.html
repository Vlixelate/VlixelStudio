In this project, I created an animation based on a provided voice-over.
First, I wrote the script in Word and broke it into clear sections.
Then, I sketched a simple storyboard in Sketchbook to plan the scenes visually.
I wanted the character to feel both loving and suffocating—using a rose with thorns to show this idea.
Writing the script helped organize the timeline.
This project was a great learning experience.
